- strong(태그) = b : 강조
- em = i : 이탤릭
- ol : ordered list // li : 리스트를 넘버링

- div != p :

                    td: data(column)

- dl : description list//dt: title//dd: definition
- blockquote : 인용구
- address : 주소 (seo:search engine optimization)
- iframe: a사이트에서 b사이트를 불러옴
